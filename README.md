# GreenNavi

GreenNaviは、太陽光発電の自家消費を最適化するためのシミュレーションツールです。
蓄電池のみを使用するシナリオと、蓄電池と水素貯蔵を組み合わせたシナリオの両方を評価し、コスト、CO2排出量、自家消費率、自給率などの主要な指標を比較分析することができます。

## 特徴

- **シナリオ比較**: 「蓄電池のみ」と「蓄電池＋水素」の2つのシナリオでシミュレーションを実行できます。
- **パラメータ設定**: 蓄電池の容量、売買電価格、水素関連設備の性能など、詳細なパラメータをGUIで設定できます。
- **結果の可視化**: シミュレーション結果をテーブル形式で表示し、主要指標や時系列データをグラフで可視化します。
- **データ入力**: 時間ごとの発電量や消費量が含まれるCSVファイルをアップロードして、手軽に分析を開始できます。

## セットアップ方法

本アプリケーションは、Dockerを使用する方法と、ローカル環境に直接セットアップする方法の2通りで利用できます。

### 1. Dockerを使用する場合 (推奨)

1.  **リポジトリをクローンします。**
    ```bash
    git clone https://github.com/your-username/GreenNavi.git
    cd GreenNavi
    ```

2.  **Dockerコンテナをビルドして起動します。**
    ```bash
    docker-compose up --build
    ```

3.  **ブラウザでアクセスします。**
    [http://localhost:8080](http://localhost:8080) を開いてください。

## 使い方

1.  **CSVファイルのアップロード**:
    サイドバーにある「CSVファイルをアップロード」ボタンから、分析したいデータをアップロードします。
    CSVファイルには、少なくとも `TIME` (日時)、`pv_net_pos_kwh` (太陽光発電量) などの列が必要です。

2.  **シミュレーション設定**:
    サイドバーで以下のパラメータを設定します。
    - **モード選択**: 「蓄電池」または「蓄電池 + 水素」を選択します。
    - **比較**: 両方のシナリオを同時に実行して比較することも可能です。
    - **各種パラメータ**: 蓄電池容量、売買電価格、水素関連設備の効率などを調整します。

3.  **シミュレーションの実行**:
    サイドバー下部の「シミュレーションを実行」ボタンをクリックすると、分析が開始されます。

4.  **結果の確認**:
    - **主要指標**: コスト、CO2排出量、自家消費率などのサマリーが表で表示されます。
    - **時系列グラフ**: 売電量、買電量、水素貯蔵量などの推移がグラフで表示されます。
    - **詳細データ**: シミュレーション結果の全データがデータフレームとして表示されます。
